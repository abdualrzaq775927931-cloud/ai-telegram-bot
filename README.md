# 📊 بوت استطلاعات الرأي والاختبارات المتطور (Advanced Poll & Quiz Bot)

بوت تيليجرام متكامل ومتقدم لإنشاء وإدارة استطلاعات الرأي والاختبارات التفاعلية، مع نظام نقاط ولوحة تحكم شاملة للمالك.

## ✨ الميزات الرئيسية
- **إنشاء استطلاعات رأي:** واجهة سهلة لإنشاء استطلاعات ومشاركتها.
- **اختبارات تفاعلية (Quizzes):** نظام اختبارات مع تقييم فوري ونقاط خبرة (XP).
- **نظام النقاط والمستويات:** تحفيز المستخدمين عبر مستويات ولوحة صدارة عالمية.
- **لوحة تحكم المالك (Super Admin):** إحصائيات النظام، إرسال رسائل جماعية (Broadcast)، وإدارة المستخدمين.
- **بنية تحتية قوية:** قاعدة بيانات SQL (دعم SQLite و PostgreSQL)، معالجة متقدمة للأخطاء، وهيكلية برمجية معيارية.
- **جاهز للنشر السحابي:** متوافق تماماً مع منصة Railway و Docker.

## 🚀 طريقة التثبيت والتشغيل

### 1. المتطلبات الأساسية
- Python 3.10+
- Telegram Bot Token (من @BotFather)

### 2. الإعداد المحلي
1. قم بتحميل الكود أو عمل Clone للمستودع.
2. قم بإنشاء بيئة افتراضية:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```
3. تثبيت المكتبات المطلوبة:
   ```bash
   pip install -r requirements.txt
   ```
4. قم بإنشاء ملف `.env` بناءً على `.env.example` وقم بتعبئة البيانات المطلوبة:
   ```env
   BOT_TOKEN=your_token_here
   ADMIN_IDS=your_telegram_id
   DATABASE_URL=sqlite:///poll_bot.db
   ```
5. تشغيل البوت:
   ```bash
   python main.py
   ```

## ☁️ النشر على Railway
المشروع جاهز للنشر المباشر على Railway:
1. قم برفع الملفات إلى مستودع GitHub خاص بك.
2. اربط مستودع GitHub بمنصة Railway.
3. أضف متغيرات البيئة (Environment Variables) في إعدادات Railway:
   - `BOT_TOKEN`
   - `ADMIN_IDS`
   - `DATABASE_URL` (سيقوم Railway بتزويدك بها عند إضافة قاعدة بيانات PostgreSQL).

## 👑 لوحة تحكم المالك
للوصول إلى لوحة التحكم، استخدم الأمر `/admin` من حساب التيليجرام الذي وضعت معرفه (ID) في متغير `ADMIN_IDS`.

## 🛠 الهيكلية البرمجية
- `bot/config/`: إعدادات البوت والمتغيرات.
- `bot/database/`: نماذج البيانات ومدير قاعدة البيانات.
- `bot/handlers/`: معالجات الأوامر والتفاعلات.
- `bot/utils/`: أدوات مساعدة إضافية.
- `main.py`: نقطة انطلاق البوت.
